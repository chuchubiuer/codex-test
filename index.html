03 / PROJECTS

Practice Process

一次公开部署，串起从本地代码到线上网页的完整过程。

  1. 01

    创建本地项目

    从简洁的 HTML、CSS 和 JavaScript 文件开始。

  2. 02

    上传 GitHub 仓库

    让代码拥有一个清晰、安全的线上版本记录。

  3. 03

    开启 GitHub Pages

    将仓库中的页面发布为可公开访问的网站。

  4. 04

    修改代码并重新部署

    每次更新都会成为页面成长的一部分。

  5. 05

    持续优化页面设计

    在内容、体验和视觉细节中不断练习。
