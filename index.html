01 / ABOUT

About This Project

这是 cc 的第一个公开上线网页练习。

目标不是一次性做完，而是通过不断修改、提交和部署，理解真实网页从本地到线上的过程。项目重点包括页面结构、视觉层级、响应式布局、交互反馈和 GitHub Pages 部署。

03 / PROCESS

From Local to Online

一次上线练习，串起完整的网页发布流程。

  1. 01

    Create Project

    创建本地网页文件。

  2. 02

    Edit Code

    修改 index.html、style.css、script.js。

  3. 03

    Commit Changes

    使用 GitHub Desktop 提交修改。

  4. 04

    Push to GitHub

    把本地代码同步到远程仓库。

  5. 05

    Deploy Pages

    开启 GitHub Pages，让网页变成公开链接。

  6. 06

    Keep Iterating

    持续优化页面设计和交互效果。
